# tic-tac-toe
A simple command line tic-tac-toe game written in JavaScript, run using NodeJS.

## To Play:
1. cd tic-tac-toe
2. npm install
3. npm start

## Synopsis:
Players 'X' and 'O' take turn inputing their position on the command line using numbers 1-9

Example board:
1 | 2 | 3
---------
4 | 5 | 6
---------
7 | 8 | 9
